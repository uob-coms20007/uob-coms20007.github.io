```c
    while (true) { 
      x = x * n; 
      y = y - 1; 
    } 
  ```

  ```python
    while true:
      x = x * n
      y = y - 1
  ```

  ```ocaml
    while true do 
      x := !x * n
      y := !y - 1
    done 
  ```

  ```ada
    while true loop
      x := x * n;
      y := y - 1;
    end loop
  ```
